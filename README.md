# Git e Github

# Aprendendo comandos git.

## 1. Iniciar repositorio

~~~bash
git init
~~~

## 2. Comando para adicionar o arquivo ao controle do Git digita o comando git e o nome do arguivo 

~~~ bash
git add Readme.md
~~~

## 3. Criar o primeiro commit (Commit e a criação de um ponto de salvamento )

~~~ bash
git commit -m 'primeiro commit'
~~~

## 4. Configurar as credenciais do Git

~~~ bash
git config user.name 'Nome_do_usuario' 
git config user.email 'Email_do_usuario'
